## [0.1.0] - 2026-05-07

### Added
- Initial UPM package release.
- YAML-driven automation framework for Unity Editor UIToolkit and IMGUI.
- 38 built-in actions (click, type, drag, assert, screenshot, etc.).
- CSS-like selector engine for VisualElement traversal.
- Headed Test Runner window with step-through debugging.
- CLI entry point for CI execution (`UnityUIFlowCliEntry.RunAllFromCommandLine`).
- Markdown/JSON reporting with screenshot-on-failure.
- Embedded `MonoHook` (MIT) for editor method interception.
- Embedded `YamlDotNet` for YAML parsing.
- Sample windows, UXML layouts, USS stylesheets, and YAML test cases.
